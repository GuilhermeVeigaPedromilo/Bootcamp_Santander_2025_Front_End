Unidades de Medidas em CSS


Unidades de medidas relativas


  • Unidade de medida em


    Os elementos tem seu tamanho relacionado com o elemento pai:
    Font-Size: 15px
    Font-Size: 30px (font-size: 2em)
    Font-Size: 22.5px (font-size: 1.5em)

  • Unidade de medida percent


    Os elementos tem seu tamanho relacionado com o elemento raiz, ou seja, o elemento html (root):
    No elemento raiz, a div possui 16px de altura
    height: 10rem (height: 160px)

  • Viewports Units (vh, vw, vmin e vmax)


    O viewport é a área visível da página no navegador. As unidades de viewport são relativas ao tamanho do viewport:
    width: 15vw
    width: 50%
    width: 10vmax (será a porcentagem do maior valor da página: comprimento ou altura)
    width: 10vmin (será a porcentagem do menor valor da página: comprimento ou altura)

  • Unidades de Medida ex e ch


    As unidades ex e ch são baseadas na altura da letra "x" minúscula e no tamanho do caractere "0", respectivamente:
    Font-Size: 15px
    Font-Size: 30px (font-size: 2ex)
    Font-Size: 22.5px (font-size: 1.5ch)

  • Estilizando com Cálculos em css ex e ch


    As unidades de medida podem ser combinadas com a função calc() para realizar cálculos dinâmicos:
    Font-Size: 15px
    Font-Size: 40px (font-size: calc(2em + 10px))
    Font-Size: 25px (font-size: calc(1.5em - 5px))
