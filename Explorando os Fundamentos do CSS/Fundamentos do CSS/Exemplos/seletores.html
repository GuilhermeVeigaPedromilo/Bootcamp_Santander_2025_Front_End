Seletores CSS

Seletores são padrões usados para selecionar os elementos HTML que você deseja estilizar.

Tipos de Seletores

  • Seletores de Tag: Seleciona todos os elementos de um tipo específico.
  • Seletores de Classe: Seleciona todos os elementos com uma classe específica.
  • Seletores de ID: Seleciona um único elemento com um ID específico.
  • Seletores Atributo: Seleciona elementos com um atributo específico.
  • Seletores Universais: Define estilos universais
