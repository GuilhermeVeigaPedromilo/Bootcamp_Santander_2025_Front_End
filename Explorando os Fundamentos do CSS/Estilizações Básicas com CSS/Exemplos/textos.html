Textos

O CSS oferece diversas maneiras de estilizar textos, permitindo controlar alinhamento, espaçamento, transformação e decoração do conteúdo textual. As principais propriedades para estilização de textos são:
  • text-align: define o alinhamento do texto (left, right, center, justify)
  • text-decoration: adiciona efeitos decorativos ao texto (none, underline, overline, line-through)
  • text-transform: controla a capitalização do texto (none, uppercase, lowercase, capitalize)
  • text-indent: define o recuo da primeira linha de um parágrafo
  • letter-spacing: define o espaçamento entre letras
  • word-spacing: define o espaçamento entre palavras
  • line-height: define a altura entre linhas de texto
  • direction: define a direção do texto (ltr para esquerda-direita, rtl para direita-esquerda)
  • white-space: controla como os espaços em branco são tratados (normal, nowrap, pre, etc.)
