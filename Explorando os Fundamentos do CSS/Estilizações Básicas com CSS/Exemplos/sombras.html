Sombras

O CSS também permite aplicar sombras para dar profundidade e destaque a elementos visuais como caixas, textos e imagens. As principais propriedades para aplicação de sombras são:
  • box-shadow: adiciona sombra a elementos em bloco (caixas). Exemplo: box-shadow: 2px 2px 6px rgba(0,0,0,0.3);
  • text-shadow: adiciona sombra ao texto. Exemplo: text-shadow: 1px 1px 2px gray;
  • filter: drop-shadow(): aplica uma sombra a imagens ou elementos inline, mantendo a forma da imagem (incluindo transparência). Exemplo: filter: drop-shadow(4px 4px 6px black);
  • filter: blur(): desfoca o elemento, podendo ser usado junto ao drop-shadow para efeitos visuais mais suaves. Exemplo: filter: blur(2px);
  • inset (com box-shadow): aplica a sombra internamente no elemento, ao invés de fora. Exemplo: box-shadow: inset 0 0 5px gray;
