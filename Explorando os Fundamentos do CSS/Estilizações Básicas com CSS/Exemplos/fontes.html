Fontes

O CSS oferece diversas maneiras de estilizar fonts, permitindo desde fonts simples até efeitos visuais mais complexos. As principais propriedades para estilização de fonts são:
  • font-family: define a família da fonte (ex: Arial, Times New Roman, sans-serif)
  • font-size: define o tamanho da fonte (ex: 16px, 1.2em, 100%)
  • font-style: define o estilo da fonte (normal, italic, oblique)
  • font-weight: define o peso da fonte (normal, bold, 100–900)
  • line-height: define a altura da linha, ou espaçamento entre linhas
  • letter-spacing: define o espaçamento entre letras
  • word-spacing: define o espaçamento entre palavras
