Fontes

O CSS oferece diversas maneiras de estilizar bordas, permitindo desde bordas simples até efeitos visuais mais complexos. As principais propriedades para controle de bordas são:
  • border-style: define o tipo de linha (solid, dashed, dotted, double, etc.)
  • border-width: define a espessura da borda
  • border-color: define a cor da borda
  • border-radius: define cantos arredondados
  • border-image: permite utilizar uma imagem como borda
  • border-style: define o tipo de linha (solid, dashed, dotted, double, outset, inset, groove, ridge, etc.)
