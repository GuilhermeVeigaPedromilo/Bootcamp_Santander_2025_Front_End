Formatando Textos com as tags

Tags <strong> e <em>

Por exemplo, você pode usar a tag strong para destacar um texto importante, e a tag em para enfatizar um texto.

Tags <sup> e <mark>

Você pode usar a tag sup para exibir texto como sobrescrito, e a tag mark para destacar um texto.

Tags <blockquote>,<u>, <i> e <b>

A tag

blockquote
é usada para citações longas, enquanto a tag u é usada para sublinhar texto. A tag i é usada para texto em itálico, e a tag b é usada para texto em negrito.
