A proprieade order: ;

A propriedade order é usada para alterar a ordem de exibição dos itens dentro de um contêiner flexível. Por padrão, todos os itens têm um valor de order igual a 0. Itens com valores de order mais baixos aparecem antes dos itens com valores mais altos.

O item 4 foi posicionado antes dos demais itens, mesmo que ele esteja definido depois dos outros.

Para isso, foi adicionado a propriedade order: 1; no item 4.

Você pode alterar o valor de order para outros itens, como por exemplo, order: 2; ou order: -1;, para ver como a ordem de exibição muda.
